# mcp-benchmark

Free, open-source CLI that estimates how much you'd save by running 
your codebase through mcp-injector before sending it to Claude.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/febin/mcp-benchmark/main/install.sh | sh
```

## Usage

```bash
mcp-benchmark ./your-project
```

## How It Works

Walks your codebase and applies two compression passes:
- **Tier 2: AST body folding** — strips function/method bodies, keeps signatures and structures intact
- **Tier 3: Comment stripping** on top of Tier 2

- **Token estimate:** bytes / 3.5 (standard heuristic for code)
- **Pricing:** Claude Sonnet at $3.00/1M input tokens (adjustable with `--price-per-million`)

**Supports:** Go, Python, TypeScript, JavaScript, Java, Rust
